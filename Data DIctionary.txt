Data Dictionary for Web-Based Lip-Reading System
This data dictionary outlines a database schema meticulously designed based on the provided UML class diagram and attributes list. It employs a Class Table Inheritance pattern, where the Accounts table stores common data for all account types, and a separate Users table stores data specific only to the user role. This approach directly translates the object-oriented inheritance model into a relational database structure.
